Web Image Editor

  • HTML5 Canvas
  • JQuerry
  • Bootstrap

Star editing by drag and drop an image from your computer. Draw on the image, crop, resize it and save the edited image to your computer.

LIVE DEMO

About

This project was build as a challange for my collage multimedia course. I wanted to learn jquery and combine it with html5 canvas to create a web interface that allows simple photo editing.

Features

Users can select to draw over the image using either predefined shapes like circles, rectangles, lines or to draw freely.

Change the image size with the posibility to keep the ratio or simply crop it to get rid of unwanted areas. Add image filters like Grayscale, Threshold, Sephia or Invert Colors.

Future Plans

This applications was build for learning purposes therefore, i do not have in plan to extend it as much because there are many good web image processing applications already.

The application is open sourced and the code is available on github.
