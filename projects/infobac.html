You don't need a computer and a notebook to learn for the baccalaureate exam.

About

The application focuses on collecting materials intended for those who want to study for the baccalaureate informatics exam, thus enabling them to learn wherever they are, all without the need for the Internet. With this application, you can monitor your progress in solving problems or exam topics, reading lessons, or solving quizzes, all for an easy exam.

Features

The app was build keeping in mind the last recomandations from google regarding architecture components. Therefore the app is using tehnologies such as DataBinding, LiveData, Room and is structured based on the ModelViewViewModel (MVVM) architecture.

Future Plans

I have in plan to add a backend service to allow users comunicate and help each other.

The application is open sourced and the code is available on github.
