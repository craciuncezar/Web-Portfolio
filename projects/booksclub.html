Book Club - Your reading companion

  • ReactJS
  • NodeJS
  • Bootstrap
  • MySQL

An web application that keeps record of your books and help you discover others.

LIVE DEMO

About

This project was build for my web technologies university course. It uses modern javascript web libraries such as ReactJS for the clientside and NodeJS and Express for the backend.

Features

The app has a navbar which separates the site in 3 areas: My Books which will hold the user created lists. Discover which will let you see awarded books and bestsellers. Search which will let you search a specific book by name.

For the authentication the app is using Passport and JSON Web Tokens. JSON Web Tokens is an authentication standard that works by assigning and passing around an encrypted token in requests that helps to identify the logged in user, instead of storing the user in a session on the server and creating a cookie.

The data about about books and authors is gathered using goodreads and google books web api. The user created lists is stored in a MySQL relational database.

Future Plans

I have in plan to create a recommendation feature which will suggest books based on the ones you saved in your lists and to fetch book reviews data from the 3rd party apis. Also i want to add support for sign in using Google or Facebook.

The application is open sourced and the code is available on github.
